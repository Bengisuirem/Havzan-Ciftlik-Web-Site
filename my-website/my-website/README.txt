Projenin Amacı:
Ailemle sahip olduğumuz işletmenin kendi web sitesini tasarlamak istedim.

Projenin içeriğinde Havzan Çiftlik Tantuni Kokoreç adlı işletmemize ait menü, iletişim bilgileri, hakkımızda bölümü ve footer kısmı mevcut.

Html, CSS ve Javascript kullanarak sitemi tasarladım.

Html'in header etiketiyle logonun ve içerklerin olduğu bölümü hazırladım.
Boyutlandırma renklendirme gibi her türlü tasarım işini CSS dosyasında yaptım.
Header kısmındaki içerikler, web sayfaları arasında geçişi sağlamak için navbar ve <a> etiketleri kullandım.

Responsive bir dizayn olması için gereken özellikleri ekledim. Bu sayede küçük bir ekranda bile kullanımı kolay olacak.

Transition kullanarak etiketler arası geçişi yavaşlatarak daha hoş bir görünüm elde ettim.

Pixel ve remçevirmesiyle uğraşmamak için font-size ayarını 62.5% olarak yaptım. bu sayede 20 pixellik bir şey yazacağımda 2 rem yazmam yeterl olacak.

Menu bölümü hazırlarken section tag'ı kullandım. Daha sonra ulaşabilmek için class ve id ekledim.
Menüdeki yemeklerin fotoğrafları, isimleri ve fiyatlarını barındırıp düzenli bir görünüm oluşturması için box-container oluşturdum.

Contact kısmına restoranımızın google map konumunu ekledim. Ve ileride sipariş oluşturmak isteyenler veya iletişime geçmek isteyenler için bir iletişime geçme paneli hazırladım.

Footer kısmında da restoranımızla ilgili ulaşabileceğiniz sosyal meyda veya diğer internet araçlarını ekledim.

JavaScript kullanarak header kısmındaki search ve menu butonlarının işilevlerini sağladım.

CSS KISMINDA KULLANDIĞIM BAZI KODLARIN İŞLEVLERİ:
border-radius: 30rem;  // Bu kullandığımız butonların veya kutucukların köşelerinin daha yuvarlak gözükmesini sağlar.

margin // Tanımlanmış sınırların dışında, etiketlerin etrafında boşluk oluşturmak için kullanılır. 

object-fit: cover; // görselin gereken alanı kaplamasını sağlar.

transform: scale(1.2); // üzerine geldiğimizde görsele zoom yapılmasını sağlar.
